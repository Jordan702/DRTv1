# DRTv1 - Decentralized Resource Token

**Mission:** Evolution Through Contribution

DRTv1 is the first open-source, contribution-based cryptocurrency aimed at revolutionizing economic systems by rewarding users with crypto for real-world acts of resource sharing and contribution — not profit.

## Key Features
- **AI-verified resource submissions**
- **Autonomous minting via OpenAI and Ethereum**
- **Transparent smart contract (verified on Etherscan)**
- **Live backend powered by Render**
- **Frontend submission portal hosted via GitHub Pages**
